# MyMacro
Macro untuk mouse biasa (player gak bermodal) dan kamu (player malas)<br/><br/>
**Fungsi :**<br/>
* F11 - Pilih Target / F12 - On Off<br/>
* Scroll Down = Search All (tombol X)<br/>
* Scroll Up = Skip (tombol ~)<br/>
* Scroll Click = 1 (tombol 1)<br/>
